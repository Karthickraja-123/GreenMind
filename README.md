# 🌱 GreenMind
Generative AI Climate & Health Intelligence Platform

## 🚨 Problem Statement
Citizens lack personalized understanding of their carbon footprint and pollution-related health risks. Most tools provide generic data without actionable AI guidance.

## 💡 Solution
GreenMind combines:
- Carbon footprint estimation (ML logic)
- Health risk prediction
- AQI-based environmental analysis
- Generative AI Climate Copilot

The system provides personalized sustainability plans and conversational climate guidance.

## 🧠 Architecture
User Input → ML Prediction (model.py) → Generative AI Reasoning (llm.py) → Streamlit Dashboard

## 🛠 Tech Stack
- Python
- Streamlit
- Machine Learning Logic
- Generative AI Simulation
- Environmental Modeling

## ▶ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run application:
streamlit run app.py

## 🌍 Impact
GreenMind transforms environmental data into personalized climate action.

ML predicts. LLM explains. People act.
